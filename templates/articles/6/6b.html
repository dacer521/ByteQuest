Functions as function arguments

Another concept that can be useful is functions that take in other functions as arguments. This can be useful for something like processing data. Let's look at an example.

In this example, the processList function did a certain behavior because it took in the processNumber function. However, if instead of adding 1.5 I wanted to add 4 to each number I could define processNumber2 and run processList with that function. Having functions take in other functions is a fairly simple, although unintuitive, concept that can be quite useful.
