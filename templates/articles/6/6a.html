What are functions

Imagine you wanted to iterate through 3 different lists and find the highest value in each of them. To do this, you’d have to write the same code 3 different times, taking up lots of space. This is a perfect example of when we would use a function. You define functions like this:

def functionName():

Function body

You then call it as functionName(). The def keyword tells python you are defining a function.
